# Chat server made in rust. 

## To interact: 

### Clone: `git clone git@github.com:SushiJ/Rust-chat-server.git`

### Run: `cargo run`

### Connection:
- Need to have [Telnet](https://www.npmjs.com/package/telnet-client) or similar client.
- Run the server
- Connect with - `telnet localhost 8080`
